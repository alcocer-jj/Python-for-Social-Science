- Johnsson, Ida Brigitta. 2022. ECON 570: Big Data Econometrics. USC Course.
- McKinney, W. (2018). Python for Data Analysis: Data wrangling with Pandas, NumPy, and IPython.
- VanderPlas, Jake. 2022. Python Data Science Handbook, 2nd Edition.
